CASE STUDY IMPLEMENTATION OF A LINKED LIST
The previous and next page in a web browser can be implemented using a linked list. The user can access the previous and next url searched in web browser by pressing the back and next button. Since webpages are linked, it can be implemented using a linked list.
The add method can be used to append a url into the linked lists of webpages.
The addAll method can be used to add a list of urls onto set of webpages
The addFirst method can be used to append a url before the head node or fisrt url of a list of linked webpages 
The addLast method can be used to append a url after the tail node or last url of a linked list of webpages
The get method can be used to search through and get a speicified url from a linked list of webpages
The remove method can be used to delete a url within the linked list of webpages
The removeFirst method can be used delete the first url i.e.  the head node from a linked list of webpages 
The removeLast method can be used to delete the last url i.e. the tail node from a linked list of webpages
The getSize method can be used to scan through all the urls of a linked list of webpages to obtain its size
Methods Implemented : - add, addAll, addFirst, addLast, get, removeFirst, removeLast, getSize

        //print the linkedList
